Front vs backend

Frontend:The client side part of a application that focuses on presenting stuff to the end-useer

Backend:The server-side part of a application that focuses on the behind-the-scenes, logic, computations, and data storage.

3 basic web technologies:

  • HTML:Markup language that describes the layout and content of a static webpage.
  • Javascript:Client side scripting language used to add dynamic functionality to a webpage.
  • CSS A language used to style a webpage.
